# 中英关键名词对照

| 中文   | 英文      | 用途                                     |
| ------ | --------- | ---------------------------------------- |
| 万界   | wanjie    | 当前游戏名称，使用一般指游戏本身相关内容 |
| 角色   | character | 一般指主角本身                           |
| 环境   | env       | 指可以切换的修炼环境                     |
| 战斗   | battle    | 与敌人对战等相关项目                     |
| 时间刻 | timetick  | 游戏运行时间                             |
| 虚空   | universe  | 指整个宇宙                               |
| 功法   | method    | 修炼功法                                 |
| 副本   | challenge | 副本及秘境等                             |
